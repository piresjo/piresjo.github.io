Billboard Chart Topper Complexity

For this project, I'm using various API calls (mainly from Billboard and Genius) to find out which songs made it to the Billboard Top 100, then with Genius and NLTK, determine the linguistic complexity of the lyrics, among other things.

Originally powered by Flask, will be updating to use Django.
